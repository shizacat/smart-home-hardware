# Description

Hardware for smart home (homebridge) on RPi Zero W.

Modules:
- Raspberry Pi Zero W
- ZigBee module from [RF start CC2652P](https://aliexpress.ru/item/1005002460600394.html)

# Hardware

The project created in [Eagle](https://www.autodesk.com/products/eagle/overview).
